## Tech Stack

 - Node
 - Express


### Libraries used  for authentication and validation 
 - zod
 - bcrypt
 - jsonwebtoken
 - bcrypt


## Run Locally

Clone the project

```bash
  git clone git@github.com:arvindbatham600/Blog-App.git
```

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node server.js
```
